# Газовый расчетник

## Описание

Интерактивное веб-приложение для прогнозирования добычи газа на основе:

- Материального баланса газовой залежи
- Двучленной формулы притока газа к скважине
- Гидравлики в стволе скважины
- Модели дожимной компрессорной станции (ДКС)
- Учета обводнения и роста водогазового фактора (ВГФ)


## Возможности
- 📝 Ввод параметров месторождения (пласт, скважины, ДКС)
- 📊 Помесячный и годовой прогноз добычи
- 📈 Интерактивные графики (Plotly)
- 💾 Экспорт результатов в Excel
- ⚙️ Настройка методов расчета (PVT, трение)

## Технологии
- Python 3.10+
- Dash / Plotly
- Pandas / NumPy
